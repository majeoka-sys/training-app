# Training App Public Safe

GitHub Pages公開用の安全版です。

- `index.html`: アプリ本体
- 過去履歴 `HD` は空
- PR `PR` は空
- 種目マスタ `MD` は種目名と標準値だけ残し、過去重量・回数・最終日などは削除

## 使い方

1. GitHubリポジトリ直下に `index.html` をアップロード
2. Settings > Pages > Deploy from a branch
3. Branch: main / root
4. 発行されたURLをiPad Safariで開く
5. ホーム画面に追加

既存データをiPadへ移したい場合は、公開HTMLには入れず、アプリ内のインポート機能でJSONをiPad SafariのlocalStorageへ入れてください。
